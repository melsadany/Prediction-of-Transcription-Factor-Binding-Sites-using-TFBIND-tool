You can use the downloaded webpage in the SRC directory for getting into the tool directly.
For getting the same results described in the report file, you should follow these instructions:
1. open the webpage
2. use the attached FASTA files for Dal80 and GAT1 to search for
3. you can choose to upload the FASTA file or write your sequence in the text box. (It's preferred to use the upload option)
4. Then, check compress results to download a file of your results.
5. After that, you should press "upload and go"

You can find the output of this tool for both genes attached in the SRC directory
